== Installation ==
* Lade den Ordner „Anfrageformular“ in das Verzeichnis /wp-content/plugins/
* Aktiviere das Plugin im Pluginmenü bei WordPress
* Klicke auf das Anfrageformular in der Menüleiste um dein erstes Formular anzusehen

Sieh dir unbedingt unseren [Quick-Start-Guide](https://anfrageformular.com/quick-start-guide-free/) an um das Anfrageformular kennenzulernen!.


== Frequently Asked Questions ==
**Was ist das Anfrageformular?**
Das Anfrageformular ist ein WordPress Plugin, mit welchem du hochwertige, verkaufsoptimiere Formulare für deine Website erstellen kannst. Neben vielen Anbindungen und Designs stehen dir nützliche weitere Funktionen zur Verfügung.

**Was bringt mir das Anfrageformular?**
Das Anfrageformular hat ein Ziel: Die Anfragen und Leads über deine Website zu steigern. Du wirst mit unserem Tool nachweislich mehr Kunden generieren. Das Tool steigert die Interaktion auf deiner Website und wird viel lieber von Besuchern ausgefüllt als ein normales Kontaktformular.

**Wie lassen sich Fragen anlegen?**
Neue Fragen lassen sich ganz einfach über unseren Drag & Drop Fragenbuilder anlegen. Du hast die Möglichkeit eine konditionale Logik, also eine aufeinander aufbauende Kette an Fragen und möglichen Antworten zu bauen (Wenn A -> Dann B).

**Welche Anbindungen gibt es?**
Die Anbindungen findest du unter der Seite Anbindungen, wir entwickeln das Plugin ständig weiter und monatlich kommen neue Integrationen hinzu.

**Ist die Einrichtung schwer?**
Nein, das WordPress Plugin ist ganz einfach und leicht zu installieren. Für jeden Einrichtungsschritt stehen dir außerdem nützliche Erklärvideos bereit. Solltest du trotzdem einmal nicht weiterkommen, steht dir unser Support gerne schnell zur Verfügung.

**Für welche Branchen ist das Anfrageformular geeignet?**
Das Anfrageformular lässt sich nahezu in jeder Branche verwenden.

**Kann ich das Anfrageformular an mein Design anpassen?**
Ja, unser Anfrageformular lässt sich 100% an deine Corporate Identity und das Design deiner Website anpassen. Alle Farben und Schriftarten sind frei konfigurierbar.

**Ist das Anfrageformular für Smartphones und Tables geeignet?**
Ja, unser Anfrageformular sieht auf jedem Gerät gut aus und ist auf alle Gerätegrößen optimal angepasst.
